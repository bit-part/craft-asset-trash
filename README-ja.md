# Asset Trash for Craft CMS 5

**[English](README.md)** | 日本語 | **[Deutsch](README-de.md)**

Craft CMS 5 用のアセットごみ箱プラグインです。アセットを削除すると、ファイルがボリューム内の `.trash/` ディレクトリにコピーされ、データベースにレコードが保存されます。コントロールパネルから復元または完全削除が可能です。

## 動作要件

- Craft CMS 5.0.0 以上
- PHP 8.2 以上

## インストール

### Composer 経由（推奨）

```bash
composer require bit-part/craft-asset-trash
```

コントロールパネルの **設定 > プラグイン** からインストールするか、CLI でインストールします：

```bash
php craft plugin/install asset-trash
```

### 手動インストール

1. [GitHub](https://github.com/bit-part/craft-asset-trash) からリリースをダウンロード
2. ディレクトリに配置し、プロジェクトの `composer.json` に [path リポジトリ](https://getcomposer.org/doc/05-repositories.md#path) を追加
3. `composer require bit-part/craft-asset-trash` を実行
4. コントロールパネルまたは CLI からインストール

## 仕組み

Craft でアセットを削除すると：

1. プラグインが削除イベントをインターセプト
2. ファイルが同じボリューム内の `.trash/` ディレクトリに**コピー**される（例：`uploads/.trash/`）
3. メタデータ（ファイル名、サイズ、ボリューム、パス、削除者、削除時のエレメント参照情報）を含むデータベースレコードが作成される
4. Craft の通常の削除処理が続行（元のアセットエレメントが削除される）

ごみ箱内のファイルは、復元または完全削除するまで `.trash/` に残ります。

## 機能

### ごみ箱一覧

コントロールパネルのメインナビゲーションに **Asset Trash** セクションが表示されます。一覧には以下が含まれます：

- ファイル名（詳細ビューへのリンク）
- ファイルサイズ
- 削除者
- 削除日時
- 削除時のエレメント参照数

### ボリュームフィルター

複数のボリュームがある場合、ドロップダウンフィルターで特定のボリュームまたは全ボリュームのごみ箱アイテムを表示できます。

### 詳細ビュー

ファイル名をクリックすると、全メタデータが表示されます：

- ファイル名、種類、ファイルサイズ
- ボリューム名と元のパス（ボリュームの基本パス＋ファイル名）
- タイトルと代替テキスト（設定されている場合）
- 削除者と削除日時
- 元のアセットエレメント ID
- ごみ箱内部パス
- 削除時のエレメント参照テーブル（ソースエレメント ID、タイプ、フィールド ID）

### 復元

ごみ箱のアイテムを元のボリュームとフォルダに復元します。プラグインが新しいアセットエレメントを作成し、ファイルを `.trash/` から元の場所に移動します。同名のファイルが既に存在する場合、一意のサフィックスが自動的に追加されます。

### 完全削除

ごみ箱のアイテムを完全に削除します。`.trash/` からファイルを削除し、データベースレコードを削除します。この操作は元に戻せません。

### 一括操作

チェックボックスで複数のアイテムを選択し、**選択した項目を復元** または **選択した項目を削除** で一括操作できます。

### ごみ箱を空にする

**ごみ箱を空にする** ボタンで、ごみ箱内のすべてのアイテム（またはフィルターで絞り込んだボリュームのアイテム）を完全に削除します。実行前に確認ダイアログが表示されます。

### 自動パージ

Craft のガベージコレクション実行時に、期限切れのアイテムを自動的にパージできます。保持期間と自動パージの有効/無効はプラグイン設定で設定します。

## 設定

**設定 > プラグイン > Asset Trash** で設定できます：

| 設定項目 | デフォルト | 説明 |
|---------|-----------|------|
| **保持日数** | `30` | 自動パージまでごみ箱のアイテムを保持する日数。`0` に設定すると無期限に保持します。 |
| **自動パージ** | `オン` | 有効にすると、Craft のガベージコレクション時に期限切れのアイテムが自動的に削除されます。 |
| **ごみ箱ディレクトリ名** | `.trash` | 各ボリューム内でごみ箱ファイルを保存するディレクトリ名。英数字、ドット、ハイフン、アンダースコアのみ使用可能です。 |

## パーミッション

**Asset Trash** の下に4つのパーミッションが登録されます：

| パーミッション | 説明 |
|--------------|------|
| **ごみ箱を表示** | コントロールパネルの Asset Trash セクションにアクセス |
| **アセットを復元** | ごみ箱のアイテムを元の場所に復元 |
| **アセットを完全削除** | ごみ箱のアイテムを個別に完全削除 |
| **ごみ箱を空にする** | 「ごみ箱を空にする」ボタンで全アイテムを一括削除 |

パーミッションはネスト構造になっており、3つのアクション用パーミッションは「ごみ箱を表示」が付与されている必要があります。
